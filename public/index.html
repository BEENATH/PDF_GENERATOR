AeroPDF REST API v1.0.0

A self-hosted PDF generation service. All endpoints require an API Key unless disabled in the server configuration.

POST
/api/v1/generate/invoice
⭐ Recommended for invoice projects. Send structured JSON — the service builds and renders the PDF automatically. No HTML knowledge needed.
fromobjectYour business info (company, email, address, phone, taxId)
toobjectClient info (name, company, email, address)
invoiceNumberstringe.g. "INV-2026-001"
invoiceDatestringe.g. "June 5, 2026"
dueDatestringe.g. "July 5, 2026"
itemsarrayRequired. [{name, description, quantity, unitPrice}]
taxRatenumberTax % e.g. 8
discountRatenumberDiscount % e.g. 5
currency / currencySymbolstringe.g. "USD" / "$"
accentColorstringBrand color hex e.g. "#2563eb"
notesstringFooter message e.g. "Payment due in 30 days"
paymentMethodstringe.g. "Bank Transfer"
downloadbooleantrue = attachment download (browser prompt)
filenamestringOutput filename e.g. "invoice-001.pdf"
POST
/api/v1/generate/html
Convert any raw HTML string into a PDF. Supports full CSS, Google Fonts, and inline styles.
htmlstringRequired. Full HTML document string.
optionsobjectformat, landscape, marginTop/Bottom/Left/Right, scale, printBackground
POST
/api/v1/generate/url
Navigate to any public URL using headless Chrome and print the rendered page as PDF. Supports JavaScript-heavy SPAs.
urlstringRequired. Full URL including https://
optionsobjectformat, landscape, margins, scale
POST
/api/v1/generate/markdown
Convert GitHub Flavored Markdown — tables, code blocks, headings, blockquotes — into a clean, typeset PDF document.
markdownstringRequired. Markdown content string.
optionsobjectformat, landscape, margins
POST
/api/v1/generate/template
Supply a Handlebars HTML template and a JSON data object. The service compiles and renders the result as PDF.
templatestringRequired. HTML with {{variable}} bindings.
dataobjectJSON variables to inject into the template.
GET
/api/v1/health
Check server health, version info, auth status, and available endpoints. No API key required.

Authentication

Include your API key in any of these ways:

Authorization: Bearer <YOUR_API_KEY>
x-api-key: <YOUR_API_KEY>
?apiKey=<YOUR_API_KEY>   (query param)
